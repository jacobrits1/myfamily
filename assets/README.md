# Assets Directory

## Logo File

Please place your logo file here as `logo.png`

**Requirements:**
- File name: `logo.png`
- Format: PNG
- Recommended size: 1024x1024 pixels
- Background: Teal/blue-green (as shown in your logo)
- Content: Three white family silhouettes

**Description of logo:**
- Circular icon
- Teal/blue-green background
- Three white human silhouettes (family)
- Left: Male figure (tallest)
- Middle: Child figure (shortest)
- Right: Female figure (tall, dress shape)

Once the logo file is placed here, run:
```bash
flutter pub run flutter_launcher_icons
```

This will generate all the required launcher icon sizes for Android and iOS.
